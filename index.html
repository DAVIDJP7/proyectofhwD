En esta web vamos a hacer diferentes guías de instalación de software de utilidad general, en los sistemas operativos de Windows y Ubuntu.

Los manuales se harán paso a paso y explicando de la manera más detallada posible a través de imágenes.

Al principio de cada software se hará una pequeña descripción del programa a instalar y en algunos casos, al final, habrá una comparativa entre dichos programas o relacionados.

Para volver a esta página, la de inicio, encontrará un dibujo de una casa al final de cada página que le traerá aquí de nuevo.
