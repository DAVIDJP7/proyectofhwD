ANTIVIRUS CLAMAV EN LINUX UBUNTU

ClamAV es un antivirus que realmente no se comporta como el típico antivirus de Microsoft Windows, que siempre está funcionando en segundo plano consumiendo recursos de memoria y CPU. Nos servirá para escanear nuestro sistema cada cierto tiempo, buscando virus.

Su instalación desde la consola de comandos, en esta distribución es:

sudo apt install clamav

×

Tras esto el antivirus ya estaría instalado y operativo en nuestro equipo, eso sí, sólo podremos usarlo desde la terminal.

Si no somos muy amantes de la línea de comandos, podemos instalar la utilidad gráfica, llamada ClamTK con el siguiente comando:

sudo apt install clamtk

×

Tras esto buscamos el nombre del programa “ClamTK” desde la barra de búsqueda de Ubuntu y encontraremos el acceso directo al programa desde la interfaz gráfica.

×

El antivirus está listo para ser usado.


COMPARATIVA AVAST VS WINDOWS DEFENDER

Avast se puede descargar de forma gratuita o se pueden elegir versiones de pago. La mayoría de los usuarios saben que Windows tiene un antivirus precargado llamado Windows Defender. Pero a pesar de esto, la mayoría de los usuarios sienten la necesidad de una seguridad adicional contra el malware y, por lo tanto, comienzan a buscar otras soluciones antivirus.


Windows Defender:

×

Proporciona seguridad en tiempo real contra ataques de virus y malware. Se ejecuta en segundo plano y proporciona a los usuarios un nivel básico de seguridad contra virus. No se puede quitar ni desinstalar a menos que se instale un producto antivirus externo. Veamos algunas de las características interesantes de Windows Defender.

Las principales ventajas de este antivirus son: Posee Cortafuegos, Las actualizaciones son automáticas, Escrutinio automatizado de archivos, Control parental y bajo consumo de recursos.

Por otro lado, algunas desventajas son: Débil alcance de la detección de malware, Actualizaciones manuales necesarias para nuevas definiciones, Sencillez a costa de variedad y lenta velocidad de escaneo.


Avast (versión gratuita):

×

Avast es una solución antivirus que está disponible para sistemas operativos como Windows, Android y Mac. Está desarrollado por Avast Software y está disponible de forma gratuita para los usuarios que deseen utilizarlo para uso personal.

Es un mito que las soluciones antivirus que están disponibles de forma gratuita no brindan protección de extremo a extremo y solo se pueden usar como un arreglo provisional.

Avast es una solución que rompe este mito. Es capaz de competir cuerpo a cuerpo con muchos de sus competidores pagados con una amplia gama de características con las que está cargado.

Las principales ventajas de este antivirus son: Tiene una versión gratuita con bastantes funcionalidades, Seguridad multifunción, Interfaz intuitivo y bajo consumo de recursos.

Desventajas de este software: Tasa de detección de virus baja, Solicitudes de actualización y ventanas emergentes con publicidad, Hay características exclusivas para planes de pago.
