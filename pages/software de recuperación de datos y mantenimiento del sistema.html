DISK DRILL DATA RECOVERY PARA WINDOWS 10

Disk drill data recovery es un software de Recuperación de Datos Gratis que sirve para recuperar rápida y fácilmente cualquier tipo de archivos borrados en Windows incluyendo documentos de Office, mensajes y archivos de media. Disk Drill para Windows es un software de recuperación de datos gratis que recupera archivos borrados desde un HDD, memoria USB o cualquier tipo de almacenamiento de media basado en Disco con un par de clicks.


Para instalar este software primero que todo vamos a ir a nuestro navegador y vamos a buscar el nombre del programa e ir a la página web oficial.


A continuación ingresamos en la página y hacemos click en el cuadrado donde pone “Descarga Gratuita”.


Damos click en el recuadro donde dice descarga gratuita y cuando esta termine abriremos el archivo descargado. Nos aparecerá este menú.


Le damos al recuadro de la derecha donde pone instalar y nos aparecerá el siguiente mensaje al que le daremos a la opción “Sí”:


Ahora nos aparecerá la barra con el porcentaje de instalación y esperaremos a que esta se complete.


Una vez terminada la instalación el programa se ejecutará automáticamente y estará listo para ser utilizado.



TESTDISK PARA UBUNTU 22.04

TestDisk es una herramienta de recuperación de datos de línea de comandos gratuita y de código abierto que se utiliza para recuperar datos de particiones eliminadas o perdidas. Además, puede usarlo para revivir particiones que no son de arranque que pueden ser causadas por factores como la eliminación accidental de tablas de particiones y ataques de malware, por mencionar algunos.


El paquete TestDisk está disponible para instalar desde los repositorios del sistema predeterminados en la mayoría de las distribuciones de Linux usando el administrador de paquetes predeterminado como se muestra.

Para comenzar, actualice los paquetes del sistema e instale TestDisk como se muestra con los siguientes comandos.


$ sudo apt update

$ sudo apt install testdisk



Para verificar que Testdisk esté instalado y mostrar más información, ejecute el siguiente comando dpkg.


$ sudo dpkg -l testdisk


El software ya está instalado. Dado que testdisk se ejecuta desde la línea de comandos, deberemos usarlo desde la terminal ya que no cuenta con una interfaz gráfica.
