CLONEZILLA

¿Qué es Clonezilla?

Clonezilla es un programa de creación/restauración de imágenes de particiones y discos completos. Es útil para desplegar sistemas de forma rápida en varios ordenadores y realizar copias de seguridad de sistemas completos.

Algunas de sus características principales son:

  • Soporte para gran cantidad de sistemas de archivos.
  • Soporte para volúmenes virtuales LVM2 en sistemas Linux.
  • Reinstalación del boot loader.
  • Soporte para discos MBR y GPT.
  • Modo desatendido.
  • Soporte para restaurar varios dispositivos simultáneamente desde una única imagen.
  • Soporte para encriptación de imágenes.
  • Soporte para copias multicast y protocolo Bittorrent.
  • Capacidad para restaurar desde dispositivo local, servidor ssh, samba NFS o WebDAV.

El software puede descargarse de manera gratuita en la web oficial del desarrollador

Una copia de seguridad mediante Samba es un método para realizar una copia de seguridad de los datos almacenados en una red a través del protocolo Samba. Samba es una implementación libre del protocolo SMB/CIFS, utilizado para compartir archivos e impresoras en redes de computadoras. Con Samba, se puede configurar un servidor de copia de seguridad que permita a los clientes de la red hacer copias de seguridad de sus datos en un servidor centralizado. La copia de seguridad se puede programar para que se realice de forma automática en intervalos regulares, lo que garantiza que los datos estén protegidos en caso de una posible pérdida o corrupción de los mismos. Además, la copia de seguridad puede ser restaurada en caso de que se produzca algún problema con los datos originales, permitiendo una recuperación rápida y eficiente de los mismos.
